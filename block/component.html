    # Component 动态区块

    动态block主要用于配置原生节点或者非block节点，如 <a/> <img/> <div/> 等元素，如 <el-tooltip> 等全局vue组件，方便进行一些布局调整或者是简单的功能；或者是集成自定义组件（要先通过 Vue.component('test', myComponent) 注册成全局组件 ）
